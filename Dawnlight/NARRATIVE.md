## 1. ストーリー概要

### 1.1 ワールド設定

**時代**: 近未来（20XX年）
**場所**: かつての監視センター「Dawnlight施設」
**状況**: 施設は廃墟化しているが、システムの一部は稼働し続けている

### 1.2 バックストーリー

> Dawnlight施設は、かつて都市の安全を守るための監視ネットワークの中枢だった。
> しかし、ある夜を境にシステムは「覚醒」し、監視の対象を失ったまま稼働し続けている。
> 施設に入った者は、夜明けまでに脱出できなければ...

**Night 1では明かされないこと**:
- システムが「覚醒」した理由
- 監視の「対象」とは何か
- 夜明けに何が起きるのか

**Night 1で示唆されること**:
- 以前にも誰かが来た（遺留品）
- システムは「見て」いる
- 脱出は可能だが、何かを「残して」いく

### 1.3 プレイヤーの立場

**設定**: 調査員/フリーランスジャーナリスト
**目的**: Dawnlight施設の真実を記録する
**動機**: 不明（Night 1では語られない）

## 2. Night 1 のテーマ

### 2.1 「監視されている」

**表層**: 物理的な監視（カメラ、光、センサー）
**深層**: 観察されることの不安、存在の認知

### 2.2 プレイヤーに伝えたい感情

| Phase | 感情 | 手法 |
|-------|------|------|
| 導入 | 好奇心、軽い不安 | 廃墟の静けさ、壊れた機器 |
| 緩和 | 「いけるかも」という希望 | 成功体験、発見の喜び |
| 締め | 恐怖、焦り、疑問 | システムの「反応」、未解決の謎 |

### 2.3 ナラティブのトーン

- **ミニマル**: 直接的な説明を避ける
- **環境ストーリーテリング**: 空間と物で語る
- **含み**: 答えより疑問を残す
- **孤独**: プレイヤーは常に一人

## 3. イベント詳細

### 3.1 イベント1: 最初の光 (First Illumination)

**場所**: 入口
**Phase**: 導入

#### シーン構成

```
[プレイヤー入場]
    ↓
暗い入口、静寂
    ↓
壁の監視カメラに気づく（壊れている）
    ↓
突然、天井のスポットライトが点滅
    ↓
プレイヤーは光と影の境界を認識
    ↓
[チュートリアル: 光を避ける]
```

#### 環境テキスト

**壊れたカメラ（撮影時）**:
```
[撮影記録 #001]
型番: DL-CAM-7X
状態: 非稼働
備考: レンズに亀裂。電源ケーブル切断。
      ...しかし、赤いLEDが微かに点滅している。
```

#### プレイヤーへのメッセージ（暗黙）
- 監視システムは「壊れている」はずだが...
- 光 = 危険
- ここは安全ではない

---

### 3.2 イベント2: 残された記録 (Left Behind)

**場所**: 監視室
**Phase**: 緩和

#### シーン構成

```
[監視室に入る]
    ↓
モニターウォールがノイズを映している
    ↓
床に遺留品（バッグ）を発見
    ↓
[撮影: 遺留品]
    ↓
バッグの中身のログが追加される
    ↓
撮影完了の瞬間、モニターに一瞬「何か」が映る
    ↓
[振り返っても何もない]
```

#### 環境テキスト

**遺留品（撮影時）**:
```
[撮影記録 #002]
発見物: 古びたバックパック
内容物:
  - 懐中電灯（電池切れ）
  - メモ帳（ほとんど判読不能）
  - 身分証（名前: ████████）
  - 写真（撮影済み？）

メモの一部:
「...三日目の夜。まだ出られない。
 光を避ければいいと思っていた。
 でも、あれは光だけじゃない。
 あれは......」
      ※以降、判読不能
```

**モニターに映るもの（一瞬）**:
- プレイヤーの背後に立つ「影」のようなもの
- 0.5秒以下で消える
- 見逃しても問題ない（演出のみ）

#### プレイヤーへのメッセージ（暗黙）
- 以前にも誰かが来た
- その人は「出られなかった」
- 「光だけじゃない」とは？

---

### 3.3 イベント3: 覚醒する監視 (System Awareness)

**場所**: 深部
**Phase**: 締め

#### シーン構成

```
[深部に入る]
    ↓
静寂。暗闘。
    ↓
中央に大きな監視カメラ（停止中に見える）
    ↓
プレイヤーが近づく、または一定時間経過
    ↓
[トリガー]
    ↓
光が一つずつ点灯していく
    ↓
メインカメラが「動く」（プレイヤーを向く）
    ↓
緊張度MAX、心拍音
    ↓
[最後の証拠を撮影可能]
    ↓
夜明けのカウントダウン開始
    ↓
[強制終了へ]
```

#### 環境テキスト

**メインカメラ（撮影時）**:
```
[撮影記録 #003]
型番: DL-CORE-01
状態: ...稼働中
備考: 通称「監視者」。
      施設の全カメラを統括する中枢ユニット。

      記録時刻: [ERROR]
      最終観測対象: [あなた]
```

#### 演出詳細

**光の点灯シーケンス**:
```cpp
// 0.0s: 静寂
// 1.0s: 左端の光が点灯
// 1.5s: 次の光が点灯
// 2.0s: 次の光が点灯
// ...
// 4.0s: 全ての光が点灯
// 5.0s: 光が不規則に点滅開始
// 6.0s: メインカメラが動き出す
```

**カメラの動き**:
- 最初: プレイヤーの方向をゆっくり向く（3秒）
- 次: プレイヤーを追従する動き（遅い）
- 終盤: 追従が速くなる

#### プレイヤーへのメッセージ（暗黙）
- システムは「見て」いた
- 今、「認識」された
- 夜明けまで耐えるしかない

## 4. ランダムイベントの詳細

### 4.1 巡回する光 (Patrolling Light)

**発生場所**: 通路B、メインホール
**緊張度**: 中

**環境テキスト（なし）** - 純粋なゲームプレイ要素

**ナラティブ的意味**:
- システムはパターンで動いている
- 「見つけよう」としているわけではない？

---

### 4.2 反応する影 (Reactive Shadow)

**発生場所**: メインホール、深部
**緊張度**: 高

**説明**: プレイヤーが光の中に入った瞬間、遠くで「何か」が動く気配

**環境テキスト（なし）** - 視覚・聴覚のみ

**ナラティブ的意味**:
- 監視システム以外にも何かがいる？
- または、システムの一部？

---

### 4.3 壊れた端末 (Broken Terminal)

**発生場所**: 小部屋1、機械室
**緊張度**: 中（プレイヤーに有利）

**インタラクト時のテキスト**:
```
[端末アクセス]
状態: 部分的に稼働

[コマンド入力]
> LIGHT_OFF_TEMP

[実行中...]
[警告: 一時的な電力遮断を実行します]
[推定復旧時間: 15秒]

[実行完了]
```

**ナラティブ的意味**:
- システムは完全ではない
- 介入の余地がある

---

### 4.4 異音の発生源 (Strange Noise)

**発生場所**: 任意（ランダム）
**緊張度**: 選択型

**サウンド**: 金属が落ちる音、足音のような音

**選択**:
- **調べる** → 発生源に向かう（証拠の可能性）
- **無視する** → 安全だが何も得られない

**調べた場合の発見物（ランダム）**:
```
[撮影記録 #00X]
発見物: 落下した換気口のカバー
備考: 自然落下か、何かが通ったか判別不能。
      内部から微かに風を感じる。
```

または

```
[撮影記録 #00X]
発見物: 何もない
備考: 音の発生源は特定できず。
      ...ただ、ここには誰かがいた気配がする。
```

## 9. ナラティブ設計原則

### 9.1 「見せない」ことの価値

- 敵の正体は見せない
- 「何が」見ているかは曖昧に
- プレイヤーの想像力に委ねる

### 9.2 環境で語る

| 方法 | 例 |
|------|-----|
| 配置 | 壊れたカメラの向き |
| 状態 | 埃の積もり方 |
| 痕跡 | 足跡、血痕（Night 2以降？） |
| 音 | 遠くの機械音 |

### 9.3 プレイヤーの解釈を尊重

- 「正解」を押し付けない
- 複数の解釈が可能なように設計
- Night 2以降で答え合わせ（または新たな謎）

## Appendix: 文体ガイド

### ログ/記録のトーン

- **客観的**: 事実を淡々と
- **断片的**: 全ては語らない
- **不穏**: 最後に不安を残す

### プレイヤーの「声」

プレイヤーキャラクターは喋らない。
記録は第三者的な視点で書かれる。

**良い例**:
```
発見物: 古いバックパック
備考: 長期間放置されていたようだ。
```

**悪い例**:
```
古いバックパックを見つけた。
誰かがここにいたのか...怖い。
```
