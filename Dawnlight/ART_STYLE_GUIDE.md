## 1. ビジュアルコンセプト

### 1.1 キーワード
- **監視** - 常に見られている不安感
- **光と影** - コントラストによる緊張
- **孤独** - 広い空間にひとり
- **冷たさ** - 無機質な環境
- **古びた近未来** - レトロフューチャー的な廃墟感

### 1.2 ムードボード参考作品

| 作品 | 参考要素 |
|------|----------|
| **Inside (Playdead)** | 光と影のコントラスト、ミニマルな色使い |
| **Limbo (Playdead)** | シルエット表現、不安感のある空間 |
| **Little Nightmares** | 歪んだパースペクティブ、巨大な空間と小さな存在 |
| **Alien: Isolation** | 監視カメラの視覚言語、工業的な環境 |
| **Control (Remedy)** | ブルータリスト建築、不気味な照明 |

### 1.3 ビジュアルテーマ: Night 1

> 「機能を失った監視システムの残骸の中を進む」

- 壊れた監視カメラ
- 点滅する蛍光灯
- 静止したモニター群
- 埃っぽい空気感

### 2.2 色の使用ルール

| 状況 | 色 | 説明 |
|------|-----|------|
| **安全エリア** | Dark Gray + Pale Green | 暗く、緑のヒント |
| **危険エリア** | Harsh White + Warning Red | 明るく、赤のアクセント |
| **中立エリア** | Mid Gray | ニュートラル |
| **検知状態** | Alert Orange → Warning Red | 段階的に赤へ |
| **隠れ場所** | Deep Black + Pale Green | 最も暗い、緑の縁取り |

### 2.3 色温度ガイド

```
安全 ←──────────────────────────────→ 危険

暖色（黄）    中性（グレー）    寒色（白/青）
 2700K          4000K             6500K+
   ↓              ↓                 ↓
古い照明      環境光           監視システム光
```

## 3. 光と影の設計

### 3.1 光源の種類

| 光源タイプ | 色温度 | 危険度 | 用途 |
|-----------|--------|--------|------|
| **監視スポットライト** | 6500K (Cold White) | 高 | プレイヤーを検知する主要光源 |
| **蛍光灯** | 4000K (Neutral) | 中 | エリア照明、点滅演出 |
| **非常灯** | 赤 | 低〜中 | 方向指示、雰囲気 |
| **モニター光** | 青 | 低 | 環境光、不気味さ |
| **古い電球** | 2700K (Warm) | 低 | 比較的安全なエリア |

### 3.2 影の表現

```
Hard Shadow（硬い影）
├─ 使用場面: 監視スポットライト
├─ 意味: 明確な境界 = 明確な危険
└─ 技術: Shadow Bias低め、シャープなエッジ

Soft Shadow（柔らかい影）
├─ 使用場面: 環境照明
├─ 意味: 曖昧な境界 = 安心感
└─ 技術: Shadow Bias高め、ソフトなエッジ
```

### 3.3 コントラスト比

- **最大コントラスト**: 安全エリア → 危険エリアの境界（1:15以上）
- **中間コントラスト**: 通常の環境照明（1:5〜1:10）
- **最小コントラスト**: 隠れ場所内部（1:2以下）

## 4. 環境アートガイドライン

### 4.1 建築スタイル

**ブルータリスト＋工業的**
- コンクリートの壁
- 露出した配管・ダクト
- 金属製の梯子・手すり
- 大きな空間と高い天井

### 4.2 環境要素

| カテゴリ | 要素 | 状態 |
|---------|------|------|
| **監視設備** | カメラ、センサー、モニター | 壊れている/機能停止 |
| **照明設備** | 蛍光灯、スポットライト | 一部点滅/故障 |
| **家具** | デスク、椅子、棚 | 放置された、埃っぽい |
| **小物** | 書類、カップ、機材 | 散乱している |
| **構造** | ドア、通気口、パイプ | 古びた、錆びた |

### 4.3 テクスチャガイドライン

```
Surface Quality (表面品質)
├─ Concrete (コンクリート): 粗い、汚れ、ひび割れ
├─ Metal (金属): 錆、傷、塗装剥がれ
├─ Glass (ガラス): 汚れ、曇り、ひび
├─ Plastic (プラスチック): 黄ばみ、傷
└─ Fabric (布): 埃、汚れ、ほつれ

Age/Wear Level: 70-80% (かなり使い込まれた状態)
```

### 4.4 スケール感

- **プレイヤーキャラクター**: 通常の人間サイズ
- **環境**: やや大きめ（1.2〜1.5倍）→ 孤独感を強調
- **天井高**: 4m〜8m（圧迫感と広大さの両立）

## 5. キャラクターアート

### 5.1 プレイヤーキャラクター

**シルエット重視**
- 顔は見せない（フード、影）
- 小さく、脆弱に見える
- 暗い色の服装

**カラー**
- ベース: #1A1A24 (Dark Gray)
- アクセント: #2A4A3A (Pale Green) - 小物

### 5.2 敵/脅威の視覚的表現

Night 1では**直接的な敵は登場しない**。脅威は光と監視システムで表現。

**監視の視覚化**
- スポットライトの動き
- カメラのレンズの反射（壊れていても）
- モニターに映る静的ノイズ

```
│  ■ UI Danger       #8B0000                          │

└─────────────────────────────────────────────────────┘
```

### 6.2 HUD設計原則

- **最小限**: 画面の5%以下を使用
- **ダイジェティック優先**: 可能な限りゲーム世界内で表現
- **透過性**: 背景が見える程度の透明度
- **位置**: 画面端、主要プレイエリアを遮らない

### 6.3 検知ゲージの視覚化

```
[安全] ░░░░░░░░░░ [危険]

段階1 (0-25%):   ░░▒░░░░░░░  - 薄い青
段階2 (25-50%):  ░░▒▒▒░░░░░  - 青 → オレンジ
段階3 (50-75%):  ░░▒▒▒▓▓░░░  - オレンジ
段階4 (75-100%): ░░▒▒▒▓▓██░  - オレンジ → 赤（点滅）
```

## 7. VFXガイドライン

### 7.1 パーティクル効果

| 効果 | 用途 | スタイル |
|------|------|----------|
| **埃** | 環境、光の中で浮遊 | 小さく、ゆっくり |
| **光の筋** | スポットライト内 | 細く、ゆっくり流れる |
| **電気スパーク** | 壊れた機器 | 青白い、短い |
| **検知パルス** | 検知時のフィードバック | 赤、外側に広がる |

### 7.2 ポストプロセス

```cpp
// Night 1 の基本ポストプロセス設定
PostProcess.Vignette = 0.4f;          // 周辺減光（強め）
PostProcess.FilmGrain = 0.15f;        // フィルムグレイン
PostProcess.ChromaticAberration = 0.1f; // 色収差
PostProcess.Bloom = 0.3f;             // ブルーム（控えめ）
PostProcess.Saturation = 0.7f;        // 彩度（低め）
PostProcess.Contrast = 1.2f;          // コントラスト（高め）
```

### 7.3 カメラ効果

| 状況 | 効果 |
|------|------|
| **通常** | 軽いビネット、フィルムグレイン |
| **検知時** | 赤みがかるビネット、画面端の歪み |
| **隠れ中** | 暗くなる、視野が狭まる（ビネット強化） |
| **撮影中** | ファインダー枠、フォーカス効果 |

## 8. アニメーションスタイル

### 8.1 キャラクターアニメーション

- **リアリスティック寄り**: 誇張しすぎない
- **緊張感**: 常に警戒している動き
- **滑らかさ**: 急な動きは検知につながるイメージ

### 8.2 環境アニメーション

| 要素 | アニメーション | 周期 |
|------|---------------|------|
| **蛍光灯の点滅** | 不規則なフリッカー | ランダム |
| **モニターノイズ** | 静的パターン変化 | 0.5秒 |
| **換気ファン** | ゆっくり回転/停止 | 連続/停止 |
| **埃の浮遊** | パーティクル | 連続 |
